Functions

LBS_Setup (host, key, secret)
设置基站定位需要登陆的服务器，验证账号和密码

Parameters:

  • host string ,LBS基站定位服务器地址
  • key string , HTTP Basic Authorization 认证用户名
  • secret string ,HTTP Basic Authorization 认证密码

Returns:

    无

Usage:

    agps.LBS_Setup("api.openluat.com","user","123345")
refresh (timeout)
下载星历数据

Parameters:

  • timeout number ,下载星历超时等待时间

Returns:

    string,星历数据的HEX字符串

Usage:

    agps.refresh(30000)
getGPD ()

Usage:

    agps.getGPD()
cellTrack (timeout)

Returns:

    string,基站坐标字符串,基站没准备好返回nil

Usage:

    agps.cellTrack()
