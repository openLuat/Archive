Functions

play (fn) 播放音频文件

Functions

play (fn)
播放音频文件

Parameters:

  • fn string ,包含路径的文件名,如果是flash内置路径为"/ldata/"

Returns:

    boole ,true 播放成功，false，播放失败

Usage:

    local let = audio.play("/ldata/call.mp3")
stop ()
停止播放当前音频文件

Returns:

    无

Usage:

    audio.stop()
play_utf8_tts (s)
播放TTS文件

Parameters:

  • s string ，UTF-8格式的字符串，支持中英文

Returns:

    无

Usage:

    audio.play_utf8_tts("欢迎光临")
play_gb2312_tts (s)
播放TTS文件

Parameters:

  • s string ，GB2312格式的字符串，支持中英文

Returns:

    无

Usage:

    audio.play_gb2312_tts("欢迎光临")
stop_tts ()
停止播放TTS文件

Returns:

    无

Usage:

    audio.stop_tts()
setChannel (chan)
设置播放通道

Parameters:

  • chan number ,播放输出通道：audiocore.HANDSET 、 audiocore.EARPIECE 、 audiocore.LOUDSPEAKER 、audiocore.BLUETOOTH 、 audiocore.FM 、 audiocore.FMLP 、 audiocore.TV 、audiocore.AUXHANSET 、 audiocore.AUXLOUDSPEAKER 、audiocore.AUXEARPIECE 、 audiocore.DUMMYHANDSET 、audiocore.DUMMYAUXHANDSET 、 audiocore.DUMMYLOUDSPEAKER 、audiocore.DUMMYAUXLOUDSPEAKER

Returns:

    boole , 设置成功返回true,失败返回false
setVolume (vol)
设置音量

Parameters:

  • vol number ,音量值为0-7

Returns:

    boole , 设置成功返回true,失败返回false

Usage:

    audio.setVolume(7)
setMicVolume (vol)
设置麦克音量

Parameters:

  • vol number ,音量值为0-15

Returns:

    boole , 设置成功返回true,失败返回false

Usage:

    audio.setMicVolume(14)
