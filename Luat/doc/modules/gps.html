Functions

open ()
打开GPS模块

Returns:

    无

Usage:

    gps.open()
close ()
关闭GPS模块

Returns:

    boole,关闭成功返回true，如果正在更新星历，则返回nil

Usage:

    gps.close()
restart (r)
重启GPS模块

Parameters:

  • r number ,重启方式-0:外部电源重置; 1:热启动; 2:温启动; 3:冷启动

Returns:

    无

Usage:

    gps.restart()
setup (id, w, m, g, vp)
设置GPS模块通信端口

Parameters:

  • id number ,串口号
  • w ，唤醒GPS模块对应的PIO
  • m ，mcu发给gps信号的PIO
  • g , GSP发给MCU信号的PIO
  • vp ，GPS的电源供给控制PIO

Returns:

    无

Usage:

    gps.setup(2,pio.P0_23,pio.P0_22,pio.P0_21,pio.P0_8)
read ()
阻塞模式读取串口数据，需要线程支持

Returns:

    返回以\r\n结尾的一行数据

Usage:

    local str = gps.read()
writeData (str)
GPS串口写数据操作

Parameters:

  • str string ,HEX形式的字符串

Returns:

    无

Usage:

    gps.writeData(str)
writeCmd (cmd)
GPS串口写命令操作

Parameters:

  • cmd string ,GPS指令(cmd格式："$PGKC149,1,115200*")

Returns:

    无

Usage:

    gps.writeCmd(cmd)
update (data)
更新星历到GPS模块

Parameters:

  • data string ,星历的十六进制字符表示字符串数据

Returns:

    boole, 成功返回true，失败返回nil

Usage:

    gps.update(data)
setAerialMode (gps, glonass, beidou, galieo)
设置搜星模式

Parameters:

  • gps number ,GPS定位系统，1是打开，0是关闭
  • glonass number ,俄罗斯Glonass定位系统，1是打开，0是关闭
  • beidou number ,中国北斗定位系统，1是打开，0是关闭
  • galieo number ，Intel 欧盟伽利略定位系统，1是打开0是关闭

Returns:

    无

Usage:

    gps.setAeriaMode(0)
fastFix (lat, lng)
秒定位命令

Parameters:

  • lat string ,参考坐标纬度
  • lng string ,参考坐标经度

Returns:

    无

Usage:

    gps.fastFix(data)
runMode (mod, rt, st)
GPS模块的运行功耗模式

Parameters:

  • mod number ,“0”，正常运行模式 “1”，周期超低功耗跟踪模式 “2”，周期低功耗模式 “4”，直接进入超低功耗跟踪模式 “8”，自动低功耗模式，可以通过串口唤醒 “9”, 自动超低功耗跟踪模式，需要force on来唤醒
  • rt number ,模式1和2的运行时间(ms),模式0的上报间隔
  • st number ,模式1和2的睡眠时间(ms)

Returns:

    无

Usage:

    gps.runMode()
